# Development Guidelines

## Code Style Preferences
- Use vanilla JavaScript for frontend (no frameworks)
- Maintain authentic Windows 95 colors and styling
- Use async/await for all API calls
- Implement comprehensive error handling
- Keep UI responsive and mobile-friendly

## Kiro Usage
- Vibe coding for Windows 95 UI generation
- Spec-driven development approach
- Agent hooks for code quality checks
- Steering documents for consistency
