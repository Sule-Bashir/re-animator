# Re-Animator Project Specification

## Project Overview
Windows 95-themed AI writing assistant that resurrects classic computing with modern AI capabilities.

## Features
- Authentic Windows 95 user interface
- AI text processing: Improve, Shorten, Summarize
- Real-time processing simulation
- Nostalgic UX with modern functionality

## Tech Stack
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Backend: Node.js, Express
- Development: Kiro for vibe coding and spec-driven development
